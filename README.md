# NpcScpSl
Плагин для управления NPC в ScpSL

# **Команды**

- **npc create(cr, spawn) [name] [classId](Default: Tutorial)** - Создает Npc
- **npc remove(rm, del) [id]** - Удаляет Npc

# **Разрешения**

- npc.create - Разрешает спавнить Npc
- npc.remove - Рарзрешает удалять Npc
